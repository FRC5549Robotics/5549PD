Chapter 2: Overview of Software

In this chapter, I will be going over all the software you need to successfully program, test, and deploy to a FRC robot. Download links will be provided as well as installation instructions.

This is the list of the core software:

  1. Latest version of Python
  2. PyCharm IDE
  3. RobotPy
  4. PyFRC
  5. PyNetworkTables
  6. RobotPy-NavX
  7. RobotPy-CTRE
  8. FRC Update Suite

  1. Latest version of Python

    At the time of writing, the latest version of Python is 3.7.3 and can be downloaded from the official Python website. We will need Python so that our programs can be interpreted once we pair it with a compatible IDE. Follow the installation guide making sure to check the option that adds Python to PATH and the option to install PIP (Python’s package manager).

  2. PyCharm IDE

    An integrated development environment (IDE), PyCharm fulfills our needs, delivering a fairly easy to use environment while providing us with features that will make coding easier such as a handy debugger and intelligent code suggestions. It can be downloaded from JetBrains website.

  3. RobotPy

    Python’s implementation of the FRC platform, RobotPy allows our code to be understood and interpreted on the robot. You do not need to perform this step if you are setting up your computer. The installation is done directly to the robot and is outlined here for explanation purposes only.

  4. PyFRC

    Installs the WPI library that contains the classes and modules FRC uses for robot programming. The installation also allows communication with a robot that has RobotPy installed on it. Follow the installation guide here.

  5. PyNetworkTables

    Special package that allows for further features by letting teams communicate with their robots through NetworkTables. Follow the installation guide here.

  6. RobotPy-NavX

    Library to communicate with NavX gyro sensors. Add it to your project by going to Files >> Settings >> Project Interpreter in PyCharm and adding RobotPy-NavX.

  7. RobotPy-CTRE

    Library to communicate with sensors and motors from Cross The Road Electronics. Add it to your project by going to Files >> Settings >> Project Interpreter in PyCharm and adding RobotPy-CTRE.

  8. FRC Update Suite

    Non-programming FRC Software Suite that contains Driver Station, RoboRIO Imaging Tool, and Radio Configuration Utility. Download it from the official National Instruments website. You will need to create a free account to be able to download the software.
